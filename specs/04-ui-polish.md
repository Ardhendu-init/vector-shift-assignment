# Part 2 — UI Polish

## Goal

Transform starter prototype into polished interview-quality workflow builder UI.

---

## Design Direction

Professional SaaS workflow builder.

Reference style:

- VectorShift
- modern AI tooling
- clean workflow editors

Desired feel:

- professional
- minimal
- cohesive
- polished

---

## Required Improvements

Improve:

- node appearance
- typography
- buttons
- spacing
- form controls
- layout consistency
- shadows
- visual hierarchy

---

## Node Styling

Nodes should feel like product components.

Requirements:

- clean cards
- consistent padding
- readable labels
- professional borders
- subtle shadows
- consistent sizing

---

## Controls

Improve:

- submit button
- text inputs
- dropdowns
- interactive controls

Requirements:

- visually consistent
- accessible enough
- modern appearance

---

## Layout

Improve:

- spacing between sections
- alignment
- workflow readability

---

## Avoid

Do NOT add:

- flashy animations
- excessive gradients
- gimmicky effects
- unnecessary complexity

---

## Acceptance Criteria

UI should feel:

- cohesive
- professional
- interview-ready
- production-inspired
